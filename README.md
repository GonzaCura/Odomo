# Odomo

Brand assets and working files for Odomo Solutions.

## Current assets

- `brand/icons/odomo-svg-icons/`: editable SVG icon family for integral business solutions.
- `brand/icons/odomo-svg-icons/icons/`: CSS-variable SVG icons.
- `brand/icons/odomo-svg-icons/icons-inline/`: SVG icons with inline colors for design tools.
- `brand/icons/odomo-svg-icons/odomo-icons-preview.svg`: vector preview sheet.
- `brand/icons/odomo-svg-icons/odomo-icons-preview.png`: raster preview sheet.
- `brand/icons/odomo-svg-icons/odomo-icons-sprite.svg`: reusable web sprite.

Palette: `#06120E`, `#F5F7F2`, `#B6FF3B`, `#00D6A3`, `#15201A`.
